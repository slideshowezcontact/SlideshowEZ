SlideshowEZ fue creado para ser un visor de diapositivas simple, rápido y directo,
reemplazando con ligereza y más herramientas al antiguo visor de diapositivas de
Windows que fue eliminado en Windows 11.

El objetivo es permitir que abras carpetas o archivos de imágenes y veas tus fotos
de inmediato, sin complicaciones y sin modificar nada en tu sistema.


CÓMO INICIAR EL SLIDESHOW
------------------------

Puedes iniciar SlideshowEZ de dos maneras:

1) Arrastrar y soltar sobre el ícono del programa
   - Arrastra una o varias carpetas y/o archivos de imágenes al ícono de SlideshowEZ
   → El slideshow comienza automáticamente con los elementos seleccionados

2) Doble clic en el ícono del programa
   - El programa abre una ventana inicial simple
   - Arrastra carpetas y/o archivos de imágenes dentro de la ventana, con o sin subcarpetas
   - Haz clic en el botón PLAY
   → El slideshow comienza con las carpetas seleccionadas


CONTROLES
---------

S / Espacio — iniciar / pausar slideshow
← →         — imagen anterior / siguiente
F           — pantalla completa
1,2,3,4,6,8 — múltiples diseños de imagen
R           — mezclar imágenes
T           — ajustar tiempo entre imágenes
Scroll      — zoom
Clic        — activar / desactivar zoom
ESC         — salir del zoom


OBSERVACIONES
-------------

• SlideshowEZ NUNCA modifica, mueve ni copia archivos o carpetas.
• La orientación EXIF se corrige automáticamente.
• Las subcarpetas se incluyen automáticamente.
