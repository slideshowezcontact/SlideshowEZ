SlideshowEZ

SlideshowEZ é um visualizador de slides simples, rápido e direto para Windows.
Ele foi criado para substituir o antigo visualizador de slides que existia no Windows
e foi removido nas versões mais recentes do sistema.

O objetivo do SlideshowEZ é permitir que você abra pastas ou arquivos de imagens
e visualize suas fotos imediatamente, sem configurações complexas, sem catálogos
e sem modificar nada no seu computador.


COMO INICIAR O SLIDESHOW
-----------------------

Você pode usar o SlideshowEZ de duas formas:

1) Arrastar e soltar sobre o ícone do programa
   - Arraste uma ou várias pastas e/ou arquivos de imagem para o ícone do SlideshowEZ
   - O slideshow inicia automaticamente com os itens selecionados

2) Abrir o programa e usar a janela inicial
   - Dê duplo clique no ícone do programa
   - O SlideshowEZ abrirá uma janela inicial simples
   - Arraste para dentro dela pastas e/ou arquivos, com ou sem subpastas
   - Clique no botão PLAY
   - O slideshow começará com os itens escolhidos


CONTROLES
---------

S / Espaço   — iniciar / pausar slideshow
← →          — imagem anterior / próxima
F            — alternar tela cheia
1, 2, 3, 4,
6, 8         — alternar layouts com múltiplas imagens
R            — embaralhar imagens
T            — ajustar tempo entre imagens
Scroll       — zoom
Clique       — ativar / desativar zoom
ESC          — sair do modo de zoom


OBSERVAÇÕES IMPORTANTES
-----------------------

• O SlideshowEZ NUNCA altera, move ou copia arquivos ou pastas.
• As imagens são apenas lidas para exibição.
• A orientação EXIF das fotos é corrigida automaticamente.
• Subpastas são incluídas automaticamente no slideshow.
• O programa não cria catálogos nem bancos de dados.
